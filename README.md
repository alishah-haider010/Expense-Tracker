 EXPENSE TRACKER (REACT-BASED FINANCE MANAGER)  
Description :  
Developed a responsive Expense Tracker web application using React 
and Vite to help users manage personal finances. The app allows users 
to add income and expense entries, view total balance, and track 
financial activity in real time. Transactions are stored using React state 
and localStorage for persistent data. Emphasized clean UI, intuitive 
interactions, and efficient state management with fast development and 
build performance powered by Vite.  
Technologies Used: React.js, CSS3, JavaScript. 
Key Concepts: useState, array manipulation, 
localStorage, conditional rendering.
